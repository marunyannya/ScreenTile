# ScreenTile
講義中、学生さんのパソコン操作状況を、画面共有により確認するアプリです。  
必要な時だけ画面共有する形になっているので、監視されている感が低く、通信量も抑えられます。  

Firestore（無料版）を利用しています。Firebaseの設定ファイル（json）を別途用意する必要があります。
詳しくは[こちら](https://marunyannya.github.io/ScreenTile/howto.html)から確認してください。
