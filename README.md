# ScreenTile
講義中、学生さんのパソコン操作状況を、画面共有により確認するアプリです。  
必要な時だけ画面共有する形になっているので、監視されている感が低く、通信量も抑えられます。  

使い方: https://marunyannya.github.io/ScreenTile/howto.html  
教　員: https://marunyannya.github.io/ScreenTile/teacher.html?config=firebase.json  
学　生: https://marunyannya.github.io/ScreenTile/student.html?config=firebase.json
